## Getting Started

### What is BK Business Plan Manager?

BK Business Plan Manager is an Obsidian plugin that helps entrepreneurs, founders, and business professionals create comprehensive business plans directly within their knowledge management system. Keep your business planning alongside your notes, research, and ideas.

### Installation

1. Open Obsidian Settings (`Cmd/Ctrl + ,`)
2. Navigate to **Community Plugins**
3. Click **Browse** and search for "BK Business Plan Manager"
4. Click **Install**, then **Enable**

*Or install manually by copying the plugin files to your `.obsidian/plugins/bk-business-plan-manager/` folder.*

## Managing Multiple Plans

### Why Multiple Plans?

You might need multiple plans for:
- Different business ideas you're exploring
- Separate plans for different product lines
- Versions for different audiences (investors vs. internal)
- Historical plans to track evolution

### Switching Plans

1. Click the **Plan Selector** dropdown in the header
2. Select the plan you want to work on
3. The interface updates to show that plan's data

### Creating Additional Plans

**From the Plans Tab:**
1. Click the **Plans** tab in navigation
2. Click the **"+ New Plan"** button
3. Choose a template and enter details

**From Settings:**

### Editing Plan Properties

1. Go to the **Plans** tab
2. Find your plan card
3. Click the **Edit** (pencil) icon

## Tips & Best Practices

### Getting Started Tips

1. **Start with the template closest to your business** - It's easier to modify than start blank
2. **Complete Executive Summary first** - It guides all other sections
3. **Don't aim for perfection** - Business plans evolve; start with basics
4. **Use the Dashboard** - Track what sections need attention

### Organization Tips

1. **One plan per business idea** - Keep ideas separate for clarity
2. **Use descriptive names** - Include year or version in name
3. **Color-code by status** - Active = green, archived = gray
4. **Regular reviews** - Set monthly calendar reminders to update

### Content Tips

1. **Be specific, not vague** - "Young professionals 25-35" not "everyone"
2. **Use data when possible** - Market size numbers, competitor stats
3. **Keep it updated** - Outdated plans lose value
4. **Link to evidence** - Reference Obsidian notes with research

### Collaboration Tips

1. **Export before meetings** - Share JSON or print sections
2. **Version control exports** - Name files with dates
3. **Separate investor version** - Create a polished plan for fundraising

## Troubleshooting

### Common Issues

#### Plugin doesn't appear
- **Check**: Is the plugin enabled in Settings â†’ Community Plugins?
- **Try**: Restart Obsidian
- **Verify**: Files exist in `.obsidian/plugins/bk-business-plan-manager/`

#### Data not saving
- **Check**: Is Auto Save enabled?
- **Try**: Manually click Save button
- **Verify**: Check `data.json` in plugin folder

#### Plan selector is empty
- **Cause**: No plans created yet
- **Solution**: Create your first plan from the welcome screen

#### Styles look wrong
- **Try**: Restart Obsidian
- **Check**: Custom CSS snippets not conflicting
- **Verify**: `styles.css` exists in plugin folder

#### Import not working
- **Check**: Valid JSON format
- **Verify**: File contains plan data structure
- **Try**: Export a plan first to see correct format

## FAQ

### General Questions

**Q: Can I use this for personal projects?**
A: Absolutely! While designed for businesses, it works great for any project planning.

**Q: Is my data stored in the cloud?**
A: No. All data is stored locally in your Obsidian vault's plugin folder (`data.json`).

**Q: Can multiple people edit the same plan?**
A: The plugin is designed for single-user use. For collaboration, export and share JSON files.

**Q: How many plans can I create?**
A: There's no limit. Create as many plans as you need.

### Data Questions

**Q: Where is my data stored?**
A: In `YOUR_VAULT/.obsidian/plugins/bk-business-plan-manager/data.json`

**Q: How do I backup my plans?**
A: Use the Export feature in Settings, or backup the `data.json` file directly.

**Q: Can I link to my Obsidian notes?**
A: Currently, plans are self-contained. Linking to notes is planned for a future update.

### Technical Questions

**Q: Does it work on mobile?**
A: Yes, the plugin works on Obsidian Mobile (iOS and Android).

**Q: Does it sync with Obsidian Sync?**
A: Yes, the `data.json` file syncs like any other plugin data.

**Q: Can I customize the appearance?**
A: The plugin follows Obsidian's theme. Custom CSS can override specific elements.
